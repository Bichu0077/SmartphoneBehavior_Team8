## Problem Statement
To discover behavioral patterns in smartphone usage data using data mining techniques.

## Objectives
- Identify user behavior groups
- Discover relationships in usage patterns

## Dataset
User Behavior Dataset from Kaggle.

## Methodology
1. Data Cleaning (missing values, outliers)
2. Exploratory Data Analysis
3. Clustering using K-Means
4. Association Rule Mining using Apriori

## Results
- Users grouped into behavioral clusters
- Strong associations found between screen time and app usage
